## Prerequisites

- [Node.js](https://nodejs.org/en/download/) (v12 or later)
- [npm](https://www.npmjs.com/get-npm) (v6 or later)

## Installation

```bash
git clone https://github.com/RiestelX/merge-sort.git
```

```bash
cd merge-sort
```

```bash
npm install
```

## Run Test

```bash
npm run test
```
